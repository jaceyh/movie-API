jMDB is a REST API that allows users to request movie data.

Users may request data such as movie descriptions, directors, and tags.
Users may also request data for particular directors and tags.
Users may register to create a personalized experience that allows them to add movies to a "Favorites" list. Users may delete their personal data at any time.
Request URL Request Method Parameters Request Format Response Format
Get list of movies /movies GET none none JSON object holding data for all movies
Get data for a single movie /movies/[name] GET :title none JSON object holding data for specified movie
Get data for a director /directors/[name] GET :director none JSON object holding data for specified director
Get data for a tag /tags/[tag] GET :tag none JSON object holding data for movies with specified tag
Create user account /users POST none JSON object holding desired username, password, email address, and birthdate

JSON object holding user data with id assigned

Get your user data /users/[Username] GET :Username JSON object holding password (eventually) JSON object holding user data

"id": 13
"username": 'Jane Doe'
"password": "xxxx1234"
"email": "xxx123@x.com"
"birthdate": "1987-02-22"

Update User Info /users/[Username] PUT :Username JSON object holding user data

{
"username": "Jane Doe"
}

JSON object containing updated user data

{
"id": "13"
"username": "Jane Doe"
"favoriteMovies": "[]"
}

Delete User Account /users/delete/[id] DELETE :id none Text response indicating user has been deleted.
Add Movie to Favorites /users/[Username]/movies/[MovieID] PUT :username, :MovieID none JSON object containing updated user data
Remove Movie from Favorites /users/[username]/movies/delete/[MovieID] DELETE :username, :MovieID none JSON object containing updated user data
